# Low-Code Platform for Dashboards & ML

Оконное приложение на PyQt6 для создания дашбордов и работы с машинным обучением. Позволяет визуально работать с данными, строить ML-модели и создавать интерактивные визуализации.

## Основные возможности
-  Предобработка данных: обработка пропусков, дубликатов, выбросов
-  Загрузка данных: CSV и JSON форматы
-  Создание дашбордов с интерактивным интерфейсом
-  ML модели: RandomForestClassifier и LinearRegression

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/usatopolosato/lowcode_platform.git
cd lowcode_platform
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
python main.py
```

## Использование

1. **Загрузка данных**: Откройте файл CSV или JSON через меню "File"
2. **Предобработка**: Используйте вкладку "Preprocessing" для очистки данных
3. **ML модели**: Выберите модель, настройте параметры и обучите
4. **Дашборд**: Создайте визуализации в конструкторе дашбордов

## Технологии

- Python 3.9+
- PyQt6 для графического интерфейса
- scikit-learn для ML моделей
- pandas для работы с данными
- matplotlib для визуализации
